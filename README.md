# S3 Manager

A simple Python script to manage S3-compatible storage.

## Features:
- List buckets
- Create a new bucket
- Delete a bucket
- Check if a bucket exists
- Upload a file from a URL
- Set object access policies
- Make a bucket public
- Get bucket policies
